## 论文写作参考

### 建议的创新点 (详见 创新点.md)

1. **SFT 数据集构建方法**: 自动化生成 25,168 条中英文混合滤波器设计训练数据
2. **QLoRA 高效微调**: 仅 0.047% 参数即可达到专业级滤波器设计能力
3. **闭环设计系统**: LLM 参数设计 → ADS 仿真验证 → 反馈优化
4. **多频段覆盖**: 统一模型处理 LPF/HPF/BPF 三种频段
5. **多轮交互设计**: 支持信息不足时追问用户，与用户交互协作
6. **反思数据生成**: 自动生成反思数据用于增强训练 (reflection)
7. **灵敏度分析**: 参数灵敏度分析数据用于增强模型理解

### 建议的论文结构

1. Introduction — RF 滤波器设计自动化的挑战与 LLM 的应用前景
2. Related Work — EDA 自动化、LLM for Science/Engineering
3. Method — 数据集构建 + QLoRA 微调 + 闭环系统架构
4. Experiments — 基线对比 (表4) + 分频段分析 (图3) + 数值精度 (图7)
5. Results & Discussion — 关键发现与改进方向
6. Conclusion
